Проект 'My Way'.
Это небольшой адаптивный сайт, сделанный при помощи Bootstrap.
Сайт посвящен теме красоте и уходу за собой, ведь в современном мире очень важно каждому выглядеть красиво и стильно.
Проект 'My Way' - сайт для студии (или салона) красоты.
На всех страницах есть адаптивная навигационная панель по сайту. Панелька имеет логотип и название студии (который меняет свой цвет при наведении). Каждая ссылка меню может изменять свой цвет. На маленьких разрешения, навигация схлопывается в маленький блок меню, при клике, список разделов выпадает, сама панелька становится черного цвета.
На главной странице есть png иконки, которые при наведении вращаются на 360deg.
На сайте присутствуют таблицы с ценовой политикой, сделанные при помощи Bootstrap.
Есть галерея с изображениями, которые изменяют свой цвет при наведении. Каждое изображение, по клику, открывается в отдельном окошке для просмотра и работает переход от одного изображения к другому.
На сайте есть информационный блок в разделе 'О Нас', который включает в себя текстовую информацию, форму обратной связи, контакты и карту расположения.
Каждый страница еще имеет Footer. Там присутствует контактная информация и иконки на социальный сети.
